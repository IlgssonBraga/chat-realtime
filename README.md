# :hammer: How to run


Clone this repo, install all dependencies and run the server:

```
$ git clone https://github.com/IlgssonBraga/chat-realtime.git
$ cd chat-realtime
[chat-realtime]$ npm install
[chat-realtime]$ npm start
```

Now, enjoy!!! :rocket::rocket:
